# Ruboty::Roll

A Ruboty plugin to roll dice.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruboty-roll'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruboty-roll

## Usage

```bash
> ruboty roll 3d6
[2, 4, 5] => 11
> ruboty roll 1d100
[41] => 41
```
